Health Care Plus



View Source Code

The project is to create an application for ‘Healthcare Plus’ hospital. This will help them take better care of patients and make their work smoother. The program can keep track of patient information quickly, help set up appointments easily, and manage things like rooms and resources well. By using this modern technology together with medical knowledge, the hospital can make better decisions, reduce mistakes, and make patients' experiences better. This project will make the hospital a leader in how it's managed, which will be good for everyone patients and the hospital staff.

The key components within the scope of this project include:
 User management
 Patient management
 Staff Management
 Record Management
 Report generating
 User Security
 Billing and Invoicing
 Medication Management

Used Methodology for this project is Waterfall Methodology

  • Caller: The caller is the person responsible for scheduling patient appointments over the phone. They can access a doctor's schedule, see existing appointments, and choose a suitable date for the patient's appointment
  • Admin: The admin is the primary user of the system and has a dashboard. Their key responsibilities include registering system users, managing patients, rooms, theaters, doctors, payments, medicines, resources, and other essential functions. Essentially, the admin oversees all the main operations of the system.

    User Requirements

  • Create Accounts to Hospital staff and login
  • Manage doctor schedules
  • Manage patient appointments
  • Manage patient records
  • Manage Doctor details
  • Manage medical records of medical resource
  • Medication Management
  • Room & theater availability
  • Billing and invoicing
  • Customizable report
